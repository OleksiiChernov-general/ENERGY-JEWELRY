# Product Orders Web App

Локальное веб-приложение для регистрации заказов на продукты.

## Запуск

```powershell
powershell -ExecutionPolicy Bypass -File ".\ENERGY JEWELRY\web-app\start-orders-web.ps1" -OpenBrowser
```

По умолчанию приложение стартует на `http://localhost:8086/`.

## Источник списка продуктов

- `ENERGY JEWELRY\CSV_Export\Лист1_csv.csv`

## Файлы результата

- `ENERGY JEWELRY\web-app\data\product-orders.json`
- `ENERGY JEWELRY\web-app\data\product-orders.xls`
